Welcome to [Hexo](http://hexo.io/)! 这里挖了一个坑，准备写点配置时候碰到的问题和解决方法。

## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Run server

``` bash
$ hexo server
```

### Generate static files

``` bash
$ hexo generate
```

### Deploy to remote sites

``` bash
$ hexo deploy
```
